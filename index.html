📸 🎥 Media Clusterer

A private-by-design media explorer. Everything runs locally — no images, videos, or data ever leave your device.

🧠 How it Works

AI models run directly in your browser via WebGPU and Transformers.js:

  • Nomic-Embed-Vision — generates 768-dim vectors for every image/video frame, in batches for speed.
  • UMAP / t-SNE / PCA (and more) — projects high-dimensional vectors onto the 2D canvas.
  • HNSW index — fast approximate nearest-neighbour search powers semantic text queries.
  • Nomic-Embed-Text — optional 134MB model for natural language search (enable in Settings).

⚡ Performance

  • Embeddings cached in IndexedDB — reload a folder instantly without re-running the model.
  • Batch size is configurable in Settings — higher values are faster but use more memory.
  • Resume button restores your last session without re-embedding.

🛡️ Privacy

Zero tracking. Zero uploads. Your folder stays local. AI models are cached in your browser for offline use.

🖱️ Navigation

  • Pan: Click and drag.
  • Zoom: Scroll wheel or pinch.
  • Open: Tap any thumbnail.
  • Search: Type a description (requires text model).
